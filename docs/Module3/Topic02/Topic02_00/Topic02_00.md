# Topic 2 - Pandas 数据处理与分析基础

本章我们将学习 Python 中一个重要的数据分析库：Pandas。Pandas 提供了强大的数据结构和数据分析工具，使得我们能够高效地处理和分析各种类型的数据。

- **Topic 2.1 - Pandas 中的数据结构**：学习 Pandas 中的两种主要数据结构：Series 和 DataFrame，以及它们的基本操作和应用。
- **Topic 2.2 - Pandas 中的数据创建与存储**：学习如何创建 Pandas 数据结构，并将数据存储到不同的文件格式中，如 CSV、Excel 等。
- **Topic 2.3 - Pandas 中的数据索引**：学习 Pandas 中的数据索引方法，包括如何按行与列进行索引，并了解多重索引结构。
- **Topic 2.4 - Pandas 中的数据基本运算**：学习 Pandas 中的基本列计算与统计计算方法，并了解使用自定义函数进行数据运算。
- **Topic 2.5 - Pandas 中的排序与聚合运算**：学习如何对数据进行排序，以及如何进行聚合计算与窗口计算。
- **Topic 2.6 - Pandas 中的数据筛选**：学习如何使用条件筛选与布尔索引来筛选数据。
- **Topic 2.7 - Pandas 中的数据清洗**：学习如何统计和处理缺失值和极端值，以及如何处理重复数据。
- **Topic 2.8 - Pandas 中的数据连接与拼接**：学习如何将多个数据集进行连接与拼接操作。
- **Topic 2.9 - Pandas 中自带的画图功能**：学习基本的 Pandas 中的绘图方法，以及 Pandas 自带图表与 Matplotlib 的关系。
